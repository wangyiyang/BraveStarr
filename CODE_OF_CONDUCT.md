# Code of Conduct

## 结论

BraveStarr 致力于建设一个尊重、专业、可协作的社区环境。

## 我们期望的行为

- 尊重不同背景、经验和观点
- 提供具体、可执行、基于事实的反馈
- 在讨论技术问题时聚焦问题本身，而不是个人
- 接受建设性批评，并愿意修正错误

## 不可接受的行为

- 人身攻击、骚扰、歧视性表达
- 恶意挑衅、持续打断或蓄意制造冲突
- 未经允许公开他人隐私、凭证或敏感信息

## 处理原则

维护者可以对不当行为采取删除内容、限制互动或拒绝协作等措施。

## 联系方式

如需报告问题，请通过仓库 Issue 或维护者公开联系方式联系。
